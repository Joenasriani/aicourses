Production-ready course content seed repository for a mobile-first AI learning platform.

- AI Content Creation Systems

## Structure

- `content/catalog.json` — machine-readable course catalog
- `content/courses/<slug>/course.json` — structured course syllabus and progression data
- `content/courses/<slug>/overview.md` — human-readable course overview

## Notes

- Each course is scoped to 5 days max.
- Free courses are beginner-first and senior-friendly where applicable.
- Content is structured for swipe/tap lessons, quizzes, mini-tasks, and a final project.
